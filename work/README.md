# Work Deliverable

本目录是比赛提交结构中的可运行交付件目录。平台加载根目录 `INSTRUCTION.md` 后，应把本目录作为作品运行资产根目录。

当前作品形态：

```text
skill/SKILL.md
skills/shophub-*.md
tools/scripts/*.py
AGENTS.md
```

不需要安装插件，不需要安装额外命令入口，也不需要把文件复制到题目仓库。运行 agent 直接读取：

1. `AGENTS.md`
2. `skill/SKILL.md`
3. `skills/*.md`
4. `tools/scripts/*.py`

然后在目标题目仓库中执行设计抽取、API 基线保护、测试诊断、代码修复、复核和报告生成。
